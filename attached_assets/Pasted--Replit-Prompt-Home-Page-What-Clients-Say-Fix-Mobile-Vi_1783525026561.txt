## MOBILE FIX (apply only at max-width: 768px)

### Problem 1 — Dark green left panel should be hidden on mobile
The left dark green panel ("THE WORD ON THE STREET / Hear what our clients have said about us!") is decorative on desktop but takes up too much space on mobile and pushes the actual review card down.

**Fix:** Hide the left panel entirely on mobile:

```css
@media (max-width: 768px) {
  .testimonial-left-panel,
  .testimonial-intro-card,
  .testimonial-green-card {
    display: none;
  }
}
```
(Use whatever class/selector targets that dark green left block)
